# haskell-import-graph

create haskell import graph for graphviz

# How to use

## Build it yourself

~~~
% git clone https://github.com/ncaq/haskell-import-graph.git
% cd haskell-import-graph
% stack install
% cd your-project-root-dir
% rm -rf .stack-work
% stack build
% stack exec haskell-import-graph
~~~
